Таблица Users:
id - Идендификатор пользователя
nickname - Никнэйм пользователя (для авторизации)
password - Пароль пользователя
surname - Фамилия
name - Имя
lastanme - Отчество
sc_degree - Учёная степень
user_articles - Статьи преподавателя

Таблица articles:
article_id - идендификатор пользователя
author - Автор(ы)
name - Имя статьи
pages - страницы публикации
article_text - текст статьи
date - дата публикации
author_id - автор опубликовавший статью
journal_id - журнал содержащий статью

Таблица journals:
journal_id - идендификатор журнала
type - тип журнала
name - имя журнала
date - дата публикации
number - номер журнала
